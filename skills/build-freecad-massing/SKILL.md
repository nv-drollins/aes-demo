# Build checked FreeCAD massing

Construct the house as editable FreeCAD solids after the site plan is approved. Use the bundled builder and specification; do not recreate eleven boxes ad hoc in the prompt.

## Prerequisites

- Require `REFERENCE_AUDIT_OK`, `SITE_SLABS_BUILD_OK`, and human approval of the site-plan rebuild.
- Keep `CliffHouseReference` immutable and build only in `CliffHouseRebuild`.
- Read `assets/cliff-house-massing-v1.json` when provenance or dimensions are being discussed.

## Procedure

1. Inspect the current site and confirm the building and garage source anchors are present.
2. Execute this exact code through FreeCAD MCP `execute_code`:

   ```python
   import os
   os.environ['AES_MASSING_SPEC'] = '/home/nvidia/aes-demo/skills/build-freecad-massing/assets/cliff-house-massing-v1.json'
   exec(compile(open('/home/nvidia/aes-demo/skills/build-freecad-massing/scripts/build-freecad-massing.py', encoding='utf-8').read(), '/home/nvidia/aes-demo/skills/build-freecad-massing/scripts/build-freecad-massing.py', 'exec'))
   ```

3. Require `FREECAD_MASSING_BUILD_OK` with exactly 11 objects.
4. Inspect every massing object with typed FreeCAD tools. Verify names, roles, groups, source references, non-zero volumes, and bounds.
5. Present fitted Front, Right, and Isometric views.
6. Explain that the boxes transcribe the public upstream `massing_v3` record. They are not inferred from the committed curve-only `.3dm`.
7. Stop for written human approval. Do not export to Blender or run ComfyUI in the same phase.

## Integrity rules

- Preserve the upstream source bounds and the documented vertical translation in the JSON spec.
- Require the imported building and garage reference bounds to match before changing the rebuild.
- Replace only objects generated by this skill's builder.
- Use only FreeCAD's built-in `Part` module.
- Treat this phase as massing: do not invent windows, doors, room layouts, railings, or cladding.
- Keep `GeneratedBy`, `MassingSpecId`, `UpstreamLayer`, `UpstreamBoundsMetres`, `BuiltBoundsMillimetres`, `SourceReferenceObjects`, and `MaterialRole` on every solid.

## Verification

Success requires the build marker, eleven solid objects, correct typed metadata, and human approval of all three views. A successful script is not approval.
